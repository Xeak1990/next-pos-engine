# Contexto del Proyecto: POS Engine
- Arquitectura: Next.js (App Router), Prisma, Tailwind CSS.
- Seguridad: JWT con doble secreto (Empleado/Cliente).
- Roles: ADMIN, MANAGER, CASHIER, CUSTOMER.
- Funcionalidades principales: Gestión de ventas, Inventario (semáforo de stock), Terminal de ventas, Reportes.
- Despliegue: Vercel (URL: https://next-pos-engine-o60cdcj7j-axel-yahir-s-projects.vercel.app)
